# Secure Password Generator

## Overview

This secure password generator is a Java-based password generator that allows users to create secure passwords by specifying password length and character requirements such as uppercase letters, lowercase letters, digits, and special characters. The program enforces user-defined rules, fills remaining characters randomly, and shuffles the final password for improved randomness. 

## Features

* Custom password length
* User-defined number of uppercase, lowercase, digit, and special characters
* Random password generation
* Password shuffling algorithm
* Console-based interface

## Concepts Used

* Java arrays
* loops
* methods
* random number generation
* ASCII character manipulation
* input validation
* swapping and shuffling algorithms
